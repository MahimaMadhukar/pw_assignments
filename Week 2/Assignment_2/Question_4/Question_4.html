1. <html>: The root element of an HTML document. It contains all other HTML elements and is usually paired with a closing </html> tag.
  2. <head>: Contains meta-information about the HTML document, such as the title, character set, and linked stylesheets.
  3. <title>: Sets the title of the HTML document, which is displayed in the browser's title bar or tab.
  4. <body>: Contains the main content of the HTML document, including text, images, links, and other elements.
  5. <h1> to <h6>: Heading elements for defining various levels of headings, with <h1> being the highest level and <h6> the lowest.
  6. <p>: Represents a paragraph of text or content.
  7. <a>: Defines a hyperlink that can link to another web page or resource.
  8. <img>: Embeds an image in the HTML document.
  9. <ul>: Creates an unordered (bulleted) list of items.
  10. <ol>: Creates an ordered (numbered) list of items.
  11. <li>: Represents a list item within an <ul> or <ol> element.
  12. <div>: A generic container for grouping and styling elements.
